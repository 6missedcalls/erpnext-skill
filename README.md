# ERPNext/Frappe Developer Skill

A Claude Code skill providing comprehensive Frappe Framework and ERPNext v15 development reference documentation.

## What This Skill Does

Gives Claude Code full context on Frappe/ERPNext APIs so it can build custom apps, DocTypes, controllers, hooks, REST APIs, form scripts, and integrations without needing to look anything up.

## Coverage

- **Framework APIs** - Document, Database, Controller, Hooks, REST API, Form Scripts
- **Module References** - Accounting, Stock, Manufacturing, CRM, HR/Payroll, Buying, Selling, Projects, Support, Assets, POS, Integrations, Setup, Customization

## Installation

Copy or symlink this directory into your Claude Code skills path, or install via the skill registry.

## Usage

Once installed, Claude Code automatically uses this skill when working on Frappe/ERPNext projects. No manual invocation needed.
